# Microsoft Fabric REST API references - Microsoft Fabric REST APIs | Microsoft Learn

Microsoft Fabric REST APIs let you automate Fabric procedures and processes.

## Overview of Microsoft Fabric REST APIs

- [Using the Microsoft Fabric REST APIs](https://learn.microsoft.com/en-us/rest/api/fabric/articles/using-fabric-apis): This page provides guidance on how to effectively utilize the Microsoft Fabric REST APIs for automation and integration tasks.
- [Fabric REST API documentation structure](https://learn.microsoft.com/en-us/rest/api/fabric/articles/api-structure): An overview of the organization and structure of the Fabric REST API documentation, helping users navigate the resources available.

## Key Concepts

- [Scopes](https://learn.microsoft.com/en-us/rest/api/fabric/articles/scopes): This section explains the concept of scopes within the Microsoft Fabric REST APIs, detailing how they are used to manage permissions and access.
- [Throttling](https://learn.microsoft.com/en-us/rest/api/fabric/articles/throttling): Information on throttling mechanisms in place to ensure fair usage of the APIs and to prevent abuse.
- [Long running operation](https://learn.microsoft.com/en-us/rest/api/fabric/articles/long-running-operation): Details on how to handle operations that take a significant amount of time to complete, including best practices for managing such requests.
- [Pagination](https://learn.microsoft.com/en-us/rest/api/fabric/articles/pagination): Guidelines on how to implement pagination when retrieving large sets of data from the APIs.
